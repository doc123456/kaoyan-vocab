# 考研英语背单词网站 - 启动说明

## 前置要求

1. **Node.js** - 版本 16 或更高
   - 下载地址：https://nodejs.org/
   - 安装后打开命令行，输入 `node -v` 验证安装

2. **npm** - 随 Node.js 一起安装
   - 输入 `npm -v` 验证安装

## 启动步骤

### 1. 安装后端依赖

```bash
cd backend
npm install
```

### 2. 初始化数据库

```bash
npm run init-db
```

这将：
- 创建 SQLite 数据库
- 创建所有必要的表
- 导入示例词库数据

### 3. 启动后端服务器

```bash
npm start
```

服务器将在 http://localhost:3001 启动

### 4. 安装前端依赖

打开新的命令行窗口：

```bash
cd frontend
npm install
```

### 5. 启动前端开发服务器

```bash
npm start
```

前端将在 http://localhost:3000 启动

### 6. 访问应用

打开浏览器访问 http://localhost:3000

## 功能说明

### 学习仪表盘
- 查看学习进度和统计数据
- 查看待复习单词数量
- 查看学习时间统计

### 传统背单词模式
- 显示单词卡片
- 点击"显示释义"查看答案
- 选择"认识"或"不认识"
- 系统根据选择调整复习间隔

### 例句模式
- 根据例句选择正确的释义
- 四选一的选择题形式
- 系统记录正确率和反应时间

## 导入更多词库

当前包含示例词库（约10个单词）。要导入完整的考研词库：

1. 从 GitHub 下载词库数据：
   - https://github.com/KyleBing/english-vocabulary
   - https://github.com/zhenghaoyang24/english-vocabulary

2. 将数据转换为项目需要的 JSON 格式

3. 替换 `backend/data/vocab_data.json` 文件

4. 重新运行 `npm run init-db`

## 常见问题

### Q: 端口被占用怎么办？
A: 修改 `backend/src/app.js` 中的 `PORT` 变量，或在启动时指定端口：
```bash
PORT=3002 npm start
```

### Q: 数据库文件在哪里？
A: 在 `backend/data/vocab.db`

### Q: 如何重置学习进度？
A: 删除 `backend/data/vocab.db` 文件，然后重新运行 `npm run init-db`

## 技术栈

- **前端**: React 18 + TypeScript + Tailwind CSS
- **后端**: Node.js + Express + SQLite
- **图表**: Recharts
- **状态管理**: Zustand
